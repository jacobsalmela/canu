# canu backup network

Canu backup network config.

```shell
canu backup network [OPTIONS]
```

### Options


### --sls-file( <sls_file>)
File containing system SLS JSON data.


### --network( <network>)
The network that is used to connect to the switches.


* **Default**

    `HMN`



* **Options**

    HMN | CMN



### --log()
enable logging.


### --folder( <folder>)
**Required** Folder to store config files


### --unsanitized()
Retain sensitive data


### --sls-address( <sls_address>)

* **Default**

    `api-gw-service-nmn.local`



### --password( <password>)
Switch password


### --username( <username>)
Switch username


* **Default**

    `admin`
